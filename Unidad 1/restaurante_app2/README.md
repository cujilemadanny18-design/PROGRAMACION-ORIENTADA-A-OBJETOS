# Sistema de Gestión de Restaurante

## Autor

Danny Cajo

## Descripción

Este proyecto fue desarrollado utilizando Programación Orientada a Objetos en Python.

El sistema permite administrar productos y clientes de un restaurante mediante una estructura modular.

## Estructura

```
restaurante_app/
│
├── modelos/
│
├── servicios/
│
├── main.py
│
└── README.md
```

## Clases

### Producto

Representa un producto del restaurante.

Atributos:

- nombre
- cantidad
- precio
- activo

### Cliente

Representa un cliente.

Atributos:

- nombre
- edad
- saldo
- vip

### Restaurante

Administra listas de productos y clientes.

## Tecnologías

- Python 3

## Cómo ejecutar

Desde la carpeta del proyecto ejecutar:

```bash
python main.py
```
